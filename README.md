# MySQL Practice

This repository contains MySQL queries created while learning SQL.
It focuses on building a strong foundation in SQL concepts used in data analysis.

## Topics Covered
- Basic SELECT statements
- WHERE conditions
- ORDER BY and LIMIT
- Aggregate functions (COUNT, SUM, AVG)
- GROUP BY and HAVING
- JOINS (INNER JOIN, LEFT JOIN)
- Subqueries

## Tools Used
- MySQL
- Ubuntu Linux
- Git & GitHub

## Repository Purpose
This project is created for hands-on SQL practice and to improve query-writing
skills required for Data Analyst and SQL-related roles.

## How to Use
- Each `.sql` file contains practice queries for a specific SQL concept
- Queries can be executed using MySQL Workbench or MySQL CLI

## Learning Status
Beginner to Intermediate level MySQL practice.
